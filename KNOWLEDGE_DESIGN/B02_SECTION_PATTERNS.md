## 8. パターン選択のガイド

### ヒーロー選択フロー
```
Q: 何を最初に見せたい？
├── サービスの画面・製品 → HERO-01, HERO-03
├── 世界観・雰囲気 → HERO-02, HERO-04
├── 情報・記事 → HERO-05
├── 参加・診断 → HERO-06
└── 作品 → HERO-07
```

### CTA選択フロー（2025年版）
```
Q: ターゲットの行動ハードルは？
├── 高い（検討期間長い）→ CTA-02（フォーム）, CTA-04（LINE）
├── 中程度 → CTA-01（バナー）, CTA-03（追従）
└── 低い（即決OK）→ CTA-05（カレンダー予約）

Q: ターゲットの年齢層は？
├── 若年層（10-30代）→ CTA-04（LINE連携）優先
└── それ以上 → CTA-02（フォーム）, CTA-05（カレンダー）
```

### 動画活用の判断
```
Q: 信頼性の担保が重要？
├── はい（BtoB、高単価）→ VIDEO-03（インタビュー動画）
├── はい（BtoC）→ VIDEO-01（UGC動画）
└── 世界観重視 → VIDEO-02（体験動画ヒーロー）
```
